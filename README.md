##Các chức năng mới
-Thêm state để quản lý dữ liệu nhập từ TextInput.
-Thực hiện validation số điện thoại khi nhấn nút "Tiếp tục" và hiển thị thông báo bằng popup.
-Thực hiện validation trong quá trình nhập, hiển thị thông báo lỗi màu đỏ nếu dữ liệu không hợp lệ.
-Tự động format dữ liệu nhập, chỉ cho phép nhập ký tự số và cập nhật lại vào TextInput
